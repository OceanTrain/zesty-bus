## 🐧 Dependancies

* npm
* cargo

## ⬇️ Install

```
cargo install wasm-pack
git clone https://github.com/OceanTrain/zesty-bus.git
cd zesty-bus/www
npm install
cd ..
```


## 🛠️ Build

```
wasm-pack build
cd www
npm run start
```

Then go to a browser and type `localhost:8080` into the url

### 🔬 Testing

```
wasm-pack test --headless --firefox
```
